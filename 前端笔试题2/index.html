正则表达式

  1. 精通使用 VUE 组件搭建前台框架,用原生 VUE.JS 或者 VUE 脚手架都可以,可以直接使用 element、antd 等组件元素；
  2. 能灵活运用 CSS3、LESS 等去绘制页面元素的样式风格,统一风格的样式要做到全局控制.
  3. 能熟练使用 echart,需要对其地图、热力图及其他常规图形组件有应用经验.
  4. 懂得使用 mock、proxy,当前先使用 mock 数据来模拟页面数据内容,api 调用要模拟后期真实数据场景状态,get、post 及其数据参数都要相对标准.当切换真实后台后要通过 proxy 或者其他方式直接切换真实 api 地址.
  5. 懂得使用 nginx 或者等同功能的 WEB 容器,部署前台页面,懂得如何使用其代理地址跳转.
  6. 精通使用 Vue 及相关技术(Vue+Vue-router+Vuex+axios+webpack);
  7. 具备前端工程化的能力,熟练使用 webpack3.x 或 4.x 及相关常见插件的配置使用;
